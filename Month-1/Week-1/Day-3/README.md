### Purpose of Directories

* `/` → Root directory of Linux
* `/home` → User files and folders
* `/etc` → Configuration files
* `/var` → Logs and variable data
* `/bin` → Essential system commands
* `/tmp` → Temporary files
* `/usr` → User applications and utilities

## What I Learned Today

* Linux follows a hierarchical file system structure.
* Everything starts from the root directory (/).
* Hidden files begin with a dot (.).
* Linux stores system configurations, logs, and commands in specific directories.
* Understanding the file system is important for server administration and troubleshooting.

## Cloud Relevance

* Cloud Engineers frequently navigate Linux servers.
* Application logs are commonly stored in `/var/log`.
* Configuration files are often located in `/etc`.
* Understanding file locations speeds up troubleshooting and server management.
