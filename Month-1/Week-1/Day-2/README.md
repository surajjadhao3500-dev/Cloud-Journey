## What I Learned Today

* Linux is primarily managed through the terminal.
* pwd shows the current directory location.
* ls displays files and folders.
* cd is used to move between directories.
* mkdir creates new directories.
* touch creates empty files.
* cat displays file contents.

## Cloud Relevance

* Cloud Engineers spend a significant amount of time working in Linux terminals.
* Most cloud servers are accessed through SSH and managed using command-line tools.
* Understanding navigation and file creation is essential for server administration.
