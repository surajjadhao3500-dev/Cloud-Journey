- Logs help identify and troubleshoot issues.
- Linux stores information about services, errors, and system activities in logs.
- journalctl is used to view systemd logs.
- Real-time log monitoring helps track ongoing system activity.
- Logs are one of the most important troubleshooting tools in Linux.

- Logs provide evidence for troubleshooting instead of guessing.
